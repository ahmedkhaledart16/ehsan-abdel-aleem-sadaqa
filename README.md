# صدقة جارية عن روح المرحومة إحسان عبد العليم

موقع ثابت كامل بالعربية RTL، جاهز للنشر على Netlify أو Vercel أو GitHub Pages.

## التشغيل محليًا
افتح ملف `index.html` مباشرة في المتصفح.

## النشر السريع
- Netlify: اسحب مجلد الموقع كاملًا داخل Netlify Drop.
- Vercel: ارفع الملفات كمشروع Static.
- GitHub Pages: ارفع الملفات على Repository وفعل Pages من Settings.

## تعديل المحتوى
- الاسم والنصوص: من ملف `index.html`.
- الألوان والتصميم: من ملف `styles.css`.
- عدادات التسبيح والقرآن والمشاركة: من ملف `script.js`.
- صورة المرحومة محسّنة داخل `assets/ehsan-abdel-aleem-enhanced.jpg` و WebP داخل `assets/ehsan-abdel-aleem-enhanced.webp`.

ملاحظة: عدادات التسبيح تُحفظ على جهاز كل زائر فقط لأنها بدون قاعدة بيانات.
